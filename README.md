# Bacaxita Headshop

Projeto frontend da Bacaxita.

## Rodar local

```bash
npm install
npm run dev
```

## Variaveis de ambiente

- `VITE_API_URL` base da API
- `VITE_ADMIN_EMAILS` emails admin separados por virgula (ex: `adm.bacaxita@gmail.com,root@bacaxita.com`)
- `VITE_GOOGLE_CLIENT_ID` client id OAuth web do Google para login
